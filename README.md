# コロシャイン

コロシャインは、落下する球を直接操作せず、塔を左右ドラッグで回転させて穴へ通し続けるスマホ向けブラウザゲームです。

公開予定URL: https://chameleonjp.codeberg.page/koroshine/

## ゲーム概要

- タイトル: コロシャイン
- 説明: くるくる回して落ち続けろ。🌰を集めて高得点を狙え。
- 対象: スマホ向けブラウザ、iPhone中心、iPhone SE対応
- 構成: `index.html` 単体公開可能
- その他のゲーム: https://chameleonjp.codeberg.page/chameleonjp_lab/

## 遊び方

1. 名前を入力します。
2. ゲーム開始後、球は自動で落下します。
3. 左右ドラッグで塔を回転させ、床の穴を球に合わせます。
4. 現在階の穴が近づくと床が光り、通過できるタイミングが見やすくなります。
5. 穴を通ると下階へ進みます。
6. 危険床に触れると短く失敗箇所を表示してからゲーム終了です。
7. 30階到達でクリアです。

## スコア

| 条件 | 加点 |
| --- | ---: |
| 1階突破 | +100 |
| 🌰取得 | +50 |
| 2連続落下コンボ | +50 |
| 3連続落下コンボ | +100 |
| 4連続落下コンボ | +150 |
| 5連続以降 | 1段ごとに +50 ずつ増加 |
| 3階以上一気落下 | +300 |
| 30階到達 | +3000 |

🌰取得時は `🌰 → ☀️` に変化し、取得位置に `☀️` と `+50` が表示されます。プレイ中は2連続以上のコンボ表示と、一気落下までの予告（`連続落下 1/3`、`あと1回で一気落下`）がCanvas上に出るため、偶然ではなく狙って加点できます。


## ユーザープレイ体験の改善

- 現在階の穴が球の真下に近づくと床が光り、通過可能な範囲ではさらに明るくなります。
- 危険床で失敗した場合は、約0.35秒だけ失敗箇所を表示してから結果へ進みます。
- Canvas上にコンボ状態と一気落下予告を表示し、`+300` の一気落下を狙いやすくしています。
- 結果画面には到達階や終了理由に応じたプレイ評価コメントが出ます。
- ランキング通信に失敗しても、長い技術エラーは表示せず、スコア・到達階・シェアはそのまま使えます。

## デザイン・操作性・読み込み改善（UI/UX）

- ゲーム開始時にCanvasの描画解像度を確定し、初回プレイでも鮮明に表示されます（初回に1×1解像度で描画される不具合を修正）。
- プレイ中以外は描画ループを停止し、モバイルのバッテリー/CPU消費を抑えます。
- ホーム追加・ステータスバー向けのメタ情報と、外部ファイル不要のインラインSVGファビコンを追加しました。
- 名前入力でEnter/完了キーから次へ進めます。
- ランキング状態を支援技術へ通知（`aria-live`）し、シェアは共有/コピー/手動コピーの順で必ず案内します。
- 詳細な役割分担レビューと採否は `docs/ui_ux_improvement_plan.md` を参照してください。

## ランキング

- 本番Supabaseランキング連携を使用します。
- Supabase URL: `https://mlpnjgezrnhdxsxolyzj.supabase.co`
- 公開HTMLに入れてよい Publishable key を使用し、secret key / service_role key は使用しません。
- `public.scores` は使用しません。
- `public.games` / `public.game_scores` / 既存RPCを利用します。
- スコア送信RPC: `submit_score`
- ランキング取得RPC: `get_best_score_ranking`
- ゲーム終了時（クリア、危険床接触、リタイア）に自動送信し、送信は1回のみです。
- 送信失敗時は短いユーザー向け文言を表示し、ローカルランキングを本番ランキング欄の代替として表示しません。スコア、到達階、結果シェアはそのまま確認できます。
